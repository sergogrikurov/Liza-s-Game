## About the Project
Math Fun Game is an educational web game designed for children in 3rd grade and above.  
It helps kids practice addition, subtraction, multiplication, and division in a fun, interactive environment.  
The game features different "rooms" with sets of math exercises where children can practice at their own pace.

This project showcases **vanilla HTML, CSS, and JavaScript**, demonstrating clean frontend coding and interactive UI design.
